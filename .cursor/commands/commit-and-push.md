# Commit and Push

## Description

Commit current changes and push to GitHub. If currently on main branch, create a new branch first. If on a feature branch, push to that branch.

## Steps

1. Check current git branch
2. If on main branch:
   - Create a new feature branch
   - Switch to the new branch
3. Stage all changes
4. Commit with appropriate message
5. Push to the current branch (new or existing)
